肖恩技术周刊（第 90 期）：过程即护城河

周刊迁移 Lumina 回顾中，待功能验证无误后切换，此站点将停止更新。大家可到 Lumina（页面顶部“信息源”链接） 回顾页面查看和订阅RSS，公众号更新不变。

在 AI 编程智能体日新月异的今天，我们很容易陷入终态焦虑。想着 AGI 终将实现，程序员迟早被替代。那现在花精力去钻研底层架构、配置复杂的开发环境、甚至是学习那些过时的 CS 基础，还有意义吗？

但实际上如果直接跳过过程去等所谓的“终点”，很可能永远也拿不到新世界的入场券。

消失的“手感”

就算 AI 能解决 80% 的重复性工作，但剩下那 20% 的极端边界、性能瓶颈和安全漏洞，才是区分“普通人”与“专家”的分水岭。

这种工程直觉无法通过阅读 AI 生成的代码来获得。会生长在每一个被手动修复的 Bug 里，生长在对比不同协议优劣的纠结里。如果没有经历这些，当 AI 生成一千行逻辑自洽但架构腐烂的代码时，根本没有能力进行有效的 Code Review。

跳过过程，意味着放弃了对系统的品鉴力，沉沦在 AI 生成的代码屎山里。

AI 需要“个人语境”

未来的编程竞争，本质上是上下文的竞争。

AI 智能体不会凭空而起，也需要扎根的土壤。我们现在折腾的每一个细节，无论是配置一套极简的 Vim 工作流，还是构建一个基于 RSS 和 Obsidian 的私有知识库。本质上都是在定义 AI 与人、与代码库交互的协议。

没有这些过程积累的环境资产，再强的模型也是外行。

成为“交响乐指挥”

开发者的角色正在从“执行者”转变为“编排者”。过去我们比拼的是手速，是代码的实现效率。未来比拼的是对复杂系统的掌控力，以及解决问题的工程思维。

这种思维的进化是不连续的，只有在“过程”的泥潭里滚过、痛过、反思过的人，才能在指挥台上站得更稳。

由此可见，过程不是抵达终点的成本，它本身就是目的。在这个代码边际成本趋向于零的时代，对工具的极端掌控感、对底层逻辑的敬畏、以及通过折腾构建出一套属于自己的工作流，才是个人真正的护城河。

终态是 AI 的，但通往终态的认知进化，永远是自己的。

个人动态

Lumina 发布 v1.1.1 版本

Lumina 是个人信息管理工作台，通过网页应用 + 浏览器扩展，帮助你完成文章采集、AI 解读和阅读管理，后续会借鉴 AK 的LLM Wiki思想持续迭代，感兴趣的朋友可以Star一下。本周发布了版本v1.1.1，主要新增以下功能：

  • AI 续写功能：支持基于 OpenAI Responses API 的原生和基于快照的续写
  • 内容引用功能：回顾编辑支持 /ref 命令快速插入文章引用
  • Markdown 导出：文章和评论详情页支持导出 Markdown

本期总结

模型层面：头部厂商从"对话工具"跃迁到"端到端生产系统"。Anthropic连发Claude Opus 4.7（长程执行/网络安全）与Claude Design（设计工作流），OpenAI、Google、阿里同步押注垂直场景，特定能力强化取代通用竞赛。

工程层面：行业开始反思，智能体的真正竞争力在"编排框架"而非模型本身，记忆机制被视为高成本负担，"AI-first"本质是流程重构。开源生态快速跟进容器化沙盒、偏见规避等标准工具。

人文张力："脑腐"批判、文科价值焦虑、安全算力化博弈，共同指向技术加速与社会适应的落差。

业界资讯

隆重推出 Claude Design

Anthropic发布Claude Design研究预览版，面向Pro、Max、Team和Enterprise用户，基于Claude Opus 4.7生成设计稿、原型、幻灯片等视觉内容，支持导入代码与文档、自动套用团队设计系统、协作编辑与导出，并可交接给Claude Code，核心影响是显著降低设计门槛、扩大探索效率并加快从想法到产品落地。

Qwen3.6-35B-A3B：智能体编程利器，现已开源

Qwen开源Qwen3.6-35B-A3B，采用35B总参数、3B激活参数的MoE架构，重点提升智能体编程与多模态推理，整体明显强于前代Qwen3.5-35B-A3B，并可对标部分27B至31B稠密模型及Claude Sonnet 4.5；模型已提供开源权重、在线体验和API接入，证明高效稀疏模型也能成为通用开源强模型。

隆重推出 Claude Opus 4.7

Anthropic正式发布Claude Opus 4.7，重点强化高难度编程、长流程自主执行、指令遵循和高分辨率视觉能力，并加入网络安全风险拦截机制；该模型已在全产品与主流云平台上线，价格不变，整体表现普遍优于Opus 4.6，但迁移时需注意更严格按提示执行及部分场景令牌消耗上升。

Gemini 3.1 Flash TTS：新一代富有表现力的人工智能语音

Google发布Gemini 3.1 Flash TTS，以低成本提供高质量文本转语音，支持70多种语言和多说话人对话，并用音频标签直接控制风格、语速与情绪切换，面向开发者、企业及Workspace用户落地全球化语音应用，同时通过SynthID水印强化AI音频识别与安全治理。

GPT-5.4-Cyber：下一代网络防御的信任访问机制

OpenAI扩大可信网络安全访问计划，向经验证的个人和团队开放更高权限，并推出为防御场景微调的GPT‑5.4‑Cyber，在加强身份验证和分级管控下提升漏洞发现、修复和逆向分析能力；其结论是让防御能力随模型能力同步扩张，广泛赋能合法防御者，同时压缩滥用风险。

佳文共赏

脑腐工业复合体

“脑腐”不是普通分心，而是平台通过高刺激、低内容的信息流持续劫持注意力、消解思考与专注能力的系统性结果，现代互联网已从传递信息转向加工用户、榨取多巴胺与利润，个体虽难立刻改变整体结构，但可通过识别触发因素、暂停无意识滑动、主动选择输入内容来夺回注意力，且互联网规则既是人造也可被重建。

AI时代，文科还有出路吗？

AI揭穿了文科大量“低阶重组与表达”能力只是高门槛信息处理，击碎“困难等于价值、训练通向判断力”的幻觉，并使学科边界与旧评价体系的利益结构暴露失效；文科若不从表达综述转向跨界整合、真实场景判断与责任承担，其能力将持续贬值并被边缘化。

网络安全如今看起来就像是工作量证明机制

Anthropic未公开发布的安全模型Mythos经第三方验证，已能在复杂企业网络攻击模拟中显著超越现有模型，且随token预算增加仍持续提升，这意味着未来系统加固本质上是与攻击者比拼谁投入更多token找漏洞，将推动开源复用更有安全优势，并让软件开发流程新增“加固”阶段，安全成本最终由漏洞市场价值决定。

懒惰的代价

作者借Larry Wall“懒惰是程序员美德”指出，优秀工程依赖为未来节省时间的抽象与简化；LLM虽显著放大写代码效率，却天然不具“懒惰”约束，易堆砌冗余、追逐代码量等虚假指标，导致系统更大更乱；LLM应仅作工具，服务人的简化判断，而非替代工程审美。

技术博客

Agent Harness 解析

智能体能力主要取决于包裹LLM的“agent harness”，而非模型本身；其核心是编排循环、工具调用、记忆、上下文管理、状态持久化、错误处理、安全护栏、验证与多智能体协作，把无状态模型变成可执行系统。生产级效果差异往往来自harness设计，优秀方向是用更薄但更稳的基础设施提升性能。

记忆可能正在损害你的AI产品

AI产品把记忆当默认卖点，但作者认为它本质是高成本的产品与系统税：常导致答案被旧偏好锚定、上下文膨胀、调试更难，并放大隐私泄露与安全投毒风险。多数场景应先做显式配置、工作流状态和任务级检索，只有长期连续型产品才值得谨慎引入记忆。

我如何用每月20美元的技术栈运营多个月10万美元收入的初创公司

作者虽有营收和用户仍因过度精简难获融资，因此给出低成本创业方案：用5到10美元VPS、Go、SQLite、本地GPU跑批量AI，必要时用OpenRouter接入顶级模型，用Copilot控开发成本；多数初创公司无需复杂云架构和风投，也能以极低烧钱实现可扩展增长。

以AI为核心不等于单纯使用AI

多数企业只是把AI嵌入原流程，效率仅小幅提升；真正的AI-first是围绕“AI主建、人类监督”重构产品、研发、测试与组织，并统一代码架构与自动化反馈闭环。作者团队借此把发布频率提升到日更级，缩短从想法到上线的周期，同时提高质量与转化，可见竞争力来自流程重构而非单点用AI。

开源项目

Impeccable：前端设计工具

Impeccable是基于Anthropic技能扩展的开源前端设计工具，内置18个AI命令与反模式库，通过7个领域参考文件指导LLM规避Inter字体、紫色渐变、卡片嵌套等常见设计偏见，支持Cursor、Claude Code等多平台即插即用，并配备独立CLI工具可扫描检测24类设计问题。

Harness：开源 DevOps 开发平台

Harness Open Source 是 Drone CI 的下一代开源 DevOps 平台，基于 Go 和 TypeScript 构建，集成代码托管、CI/CD 流水线、Gitspaces 开发环境及制品仓库，支持 Docker 本地部署，旨在提供端到端的开源开发体验。

OpenAI Agents SDK：轻量级多智能体工作流框架

OpenAI Agents SDK是面向多智能体工作流的轻量框架，兼容OpenAI及100多种LLM，提供智能体配置、工具调用、安全校验、人机协作、会话管理、追踪调试和实时语音能力，0.14.0新增容器化沙盒智能体，可在本地执行长时任务、检查文件、运行命令和打补丁，提升复杂自动化任务的可控性与扩展性。

architecture-diagram-generator：基于Claude技能的架构图生成工具

该工具面向Claude AI，把自然语言系统架构描述自动生成可独立打开的深色主题HTML/SVG架构图，用户安装技能后用文本描述组件与连接即可快速出图并对话式迭代修改，最终降低架构图制作门槛，支持分享、打印、导出PDF和静态托管，适用于多种常见架构场景。

学习资料

《动手学大模型》

《动手学大模型》是上海交大张倬胜团队推出的免费实践教程，面向初学者提供大模型入门与课程研究支持，覆盖微调部署、提示学习、知识编辑、数学推理、多模态、智能体与安全对齐等11个主题，并配套课件、代码和脚本；2025年6月又新增与华为昇腾联合开发的国产化《大模型开发全流程》教程，形成更完整的入门到进阶学习体系。

Cursor 学习教程

Cursor Learn 是一门面向开发者的实用课程，核心是教你理解 AI 模型的原理、类型与局限，并用 Cursor 等工具提升写代码效率。课程强调在成本、速度和可靠性间做取舍，建立正确心智模型，避免生成代码不可用的低效循环，最终掌握可迁移到各类 AI 工具的软件开发方法。

人工智能指数报告

AI能力、普及和产业化正以快于治理、评估、教育和基础设施的速度扩张，2025年产业主导前沿模型，中美顶尖模型差距缩至2.7%，生成式AI三年覆盖过半人口，就业冲击、安全事故、透明度下降与资源消耗同步加剧。技术突破已领先社会适应，全球进入寡头化竞争与治理滞后并存阶段。
